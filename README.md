# fgt

- This has been tested against php 5.5.12
- Have composer installed https://getcomposer.org/download/
- Clone this project
- Run the program
``` 
    cd bin 
    ./program < file or ./program filepath 
```
- To run phpunits
```
    phpunit -c phpunit.xml
```
